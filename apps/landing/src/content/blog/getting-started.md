Welcome to Rwote! This guide will help you get started capturing insights from your learning sessions.

## What is Rwote?

Rwote is a simple tool for capturing and organizing insights while you learn. Whether you're reading documentation, watching tutorials, or taking courses, Rwote helps you save the important things you discover.

## Key Features

- **Chrome Extension**: Save insights with one click while browsing
- **Tags**: Organize notes by topic for easy filtering
- **Cloud Sync**: Access your notes on any device

## Getting Started

1. Sign up for a free account
2. Install the Chrome extension
3. Start capturing insights!
